= ROS2 Action API Reference

[[action-dori-nav-navigate_to]]
=== /nav/navigate_to

*Type (action)*:: `navigation_interfaces/action/Navigate`
*Servers*:: navigator_node
*Clients*:: llm_node (or other HRI/LLM orchestration clients)
*Flow*:: llm_node → /nav/navigate_to (goal/cancel) → navigator_node → feedback/result to llm_node

.Goal
[cols="1,1,1,3",options="header"]
|===
| Field | Type | Unit | Description
| destination | geometry_msgs/msg/PoseStamped | - | Navigation target pose.
|===

.Feedback
[cols="1,1,1,3",options="header"]
|===
| Field | Type | Unit | Description
| distance_remaining | float32 | m | Remaining distance to goal.
| state | string | - | Internal navigator state enum name.
| status_message | string | - | Human-readable progress/status text.
|===

.Result
[cols="1,1,1,3",options="header"]
|===
| Field | Type | Unit | Description
| code | int32 | - | `SUCCESS`, `CANCELED`, `PLANNING_FAILED`, or `INVALID_STATE`.
| message | string | - | Result detail message.
|===
