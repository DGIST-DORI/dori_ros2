= ROS2 Parameter API Reference

This is a single source (SSOT) for ROS2 parameter details documentation.

* Parameter anchors use the `param-...` format.
* Parameters are grouped by node.
* NOTE: Use `-` for unitless parameters. Prefer SI units (`s`, `m`, `Hz`, `rad`) unless domain conventions require otherwise.
