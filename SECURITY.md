# Security Policy

## Reporting a Vulnerability

If you discover a vulnerability, please report it via email:

jaewon1627@gmail.com

Do NOT create a public GitHub issue.

## Response Time

- Acknowledge within 48 hours
- Fix or mitigation within 7 days

## Supported Versions

We currently support only the latest version.

## Disclosure Policy

Please keep vulnerabilities private until they are fixed.
We appreciate responsible disclosure and will
credit reporters if they wish.
